# 🎯 Study Grinder

### Built With
JavaScript / Node.js
Electron framework
Native HTML5 / CSS3 Grid-Flexbox
Press Start 2P Google Web Fonts

## Features
* **Frameless & Transparent UI:** Sits beautifully floating on your desktop wallpaper.
* **Snappy Play/Pause Controls:** Fast interactive click detection wrapper.
* **Smart Daily Streak Tracker:** Increments your focus streak once a day when you finish a session. Resets if a calendar day is missed entirely!
* **Daily Session Record:** Keeps track of your largest continuous study session per day using `localStorage`.

## How to Run Locally

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your computer.

### Installation
1. Clone this repository or download the project files.
2. Open your terminal/command prompt inside the project folder.
3. Install the required Electron dependency packages by running:
   ```bash
   npm install

### Launching the Widget
To boot up your pixel widget application live on your screen, run:
   ```bash
   npm start
